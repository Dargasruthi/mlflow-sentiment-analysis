# MLflow Sentiment Analysis Project

## Description
This project demonstrates the use of MLflow for experiment tracking and model management in a sentiment analysis machine learning task.

## Tools Used
- Python
- Scikit-learn
- MLflow
- VS Code

## Features
- Experiment tracking
- Parameter and metric logging
- Model artifact storage
- MLflow UI visualization

## How to Run
1. Install requirements  
   `pip install mlflow pandas scikit-learn`
2. Run MLflow UI  
   `python -m mlflow ui`
3. Run training script  
   `python train.py`
